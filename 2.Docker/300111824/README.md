## ce docker file permet d'afficher ce qui se trouve dans un fichier **.txt**
---------------------------------------------------------------------

### :a: pour creer l'image

```
docker build --tag blevie:1.0 .

```

### :b: pour executer ce ficher
```
docker run --name bleviecontainer blevie:1.0

```
